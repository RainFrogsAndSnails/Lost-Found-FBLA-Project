# Lost-and-Found Website - Full-Stack Learning Checklist

| Phase                                 | Skill / Task                                                            | Completed (✔) |
| ------------------------------------- | ----------------------------------------------------------------------- | ------------- |
| **Phase 1: HTML & CSS**               | Create HTML structure for 5 pages (Home, Submit, Search, Claim, Admin)  |               |
|                                       | Add navigation bar linking all pages                                    |               |
|                                       | Use semantic HTML (`header`, `nav`, `main`, `footer`)                   |               |
|                                       | Style pages with CSS: fonts, colors, spacing, borders                   |               |
|                                       | Layout pages using Flexbox & Grid                                       |               |
|                                       | Add hover effects and buttons                                           |               |
|                                       | Make pages responsive with `@media` queries                             |               |
| **Phase 2: Basic JavaScript**         | Learn variables (`let`, `const`)                                        |               |
|                                       | Learn arrays & objects                                                  |               |
|                                       | Learn functions (`function`)                                            |               |
|                                       | Capture form inputs with DOM (`getElementById`, `querySelector`)        |               |
|                                       | Add event listeners (`addEventListener`)                                |               |
|                                       | Prevent default form submission (`e.preventDefault()`)                  |               |
|                                       | Store form data in an array                                             |               |
|                                       | Display array data dynamically on the page (`innerHTML`, `appendChild`) |               |
| **Phase 3: Advanced JS & Frontend**   | Edit/delete items from array                                            |               |
|                                       | Mark items as claimed                                                   |               |
|                                       | Add simple admin actions (approve/remove)                               |               |
|                                       | Validate input fields (required, email, etc.)                           |               |
|                                       | Advanced CSS styling for forms and lists                                |               |
|                                       | Accessibility: alt text, readable contrast, labels                      |               |
|                                       | Implement live search/filter on items                                   |               |
| **Phase 4: Backend Basics**           | Setup Node.js project (`npm init`)                                      |               |
|                                       | Install and use Express to serve pages                                  |               |
|                                       | Handle routes (`app.get()`, `app.post()`)                               |               |
|                                       | Setup SQLite or MySQL database                                          |               |
|                                       | Create tables and insert/select/delete items                            |               |
|                                       | Connect backend to store and fetch submissions                          |               |
| **Phase 5: Full-Stack Integration**   | Connect form submissions → backend → database → display                 |               |
|                                       | Implement admin dashboard CRUD functionality                            |               |
|                                       | Test full workflow: submit, search, claim, approve                      |               |
| **Phase 6: Polishing & Presentation** | Test responsiveness on mobile, tablet, desktop                          |               |
|                                       | Test accessibility: alt text, ARIA labels, contrast                     |               |
|                                       | Clean code, add comments                                                |               |
|                                       | Add graphics/icons purposefully                                         |               |
|                                       | Prepare presentation: explain code & demo features                      |               |
