# Lost-and-Found Website - Full-Stack Learning Checklist

## Phase 1: HTML & CSS

- [ ] 5-page HTML structure (Home, Submit, Search, Claim, Admin)
- [ ] Navigation bar & semantic HTML
- [ ] CSS: fonts, colors, spacing, borders
- [ ] Flexbox & Grid layouts
- [ ] Hover effects & buttons
- [ ] Responsive design (`@media` queries)

## Phase 2: Basic JS

- [ ] Variables (`let`, `const`)
- [ ] Arrays & objects
- [ ] Functions
- [ ] DOM: `getElementById()`, `querySelector()`
- [ ] Event listeners & `preventDefault()`
- [ ] Store form data in arrays
- [ ] Display data dynamically (`innerHTML`, `appendChild`)

## Phase 3: Advanced JS & Frontend

- [ ] Edit/delete array items
- [ ] Mark items as claimed
- [ ] Admin actions (approve/remove)
- [ ] Input validation (required, email, etc.)
- [ ] Advanced CSS for forms & lists
- [ ] Accessibility (alt text, contrast, labels)
- [ ] Live search/filter functionality

## Phase 4: Backend Basics

- [ ] Node.js setup (`npm init`)
- [ ] Express server & page serving
- [ ] Routes (`app.get()`, `app.post()`)
- [ ] SQLite or MySQL database
- [ ] CRUD operations on database
- [ ] Backend ↔ database connection

## Phase 5: Full-Stack Integration

- [ ] Form → backend → database → display
- [ ] Admin dashboard CRUD
- [ ] Full workflow testing

## Phase 6: Polishing & Presentation

- [ ] Responsive testing (mobile/tablet/desktop)
- [ ] Accessibility testing (ARIA, contrast)
- [ ] Clean code & comments
- [ ] Graphics/icons
- [ ] Presentation prep
